Ejercicio 5.
Dado un alfabeto Σ, sean 𝑥, 𝑦 ∈ Σ y 𝛼, 𝛽 ∈ Σ∗. Demostrar que:
a. |𝑥.(𝑦.𝛼)| = 2 + |𝛼|
Por inducción estructural sobre 𝛼:

  • Caso base: 𝛼 = 𝜆
    |𝑥.(𝑦.𝛼)| = |𝑥.(𝑦.𝜆)| = |𝑥.𝑦| = 2 = 2 + 0 = 2 + |𝜆| = 2 + |𝛼|

c. |𝛼𝑥𝛽| = |𝑥𝛼𝛽|

  • Caso base: 𝛼 = 𝜆

d. |𝛼.𝛼| = 2|𝛼|
Por inducción estructural sobre 𝛼:

  • Caso base: 𝛼 = 𝜆
    |𝛼.𝛼| = |𝜆.𝜆| = |𝜆| = 0 = 2 * 0 = 2|𝜆| = 2|𝛼|
  • Caso inductivo: 𝛼 = 𝛼´.𝑥. Supongo que vale para 𝛼´.

e. ℒ = {𝛼₁𝑏𝛼₂ | 𝛼₁, 𝛼₂ ∈ {𝑎, 𝑏}∗ ∧ |𝛼₁| > |𝛼₂|} para Σ = {𝑎, 𝑏}

Ejercicio 10.
Sea ℒ, ℒ₁, ℒ₂ lenguajes cualesquiera. Determinar si las siguientes afirmaciones son verdaderas o falsas. Si son verdaderas, demostrarlas. Si no, dar un contraejemplo.
a. ℒ ⁺ ⊆ ℒ ⁺

Verdadera. Todo conjunto es subconjunto de sí mismo.

b. ℒ ⁺ ⊊ ℒ ∗

Falso. Contraejemplo: ℒ = {𝜆, 𝑎}. {𝜆, 𝑎}⁺ = {𝜆, 𝑎}∗.

c. ℒ ⁿℒ ᵐ = ℒ ⁿ⁺ᵐ para todo 𝑛, 𝑚 ≥ 0

Veradero. Por inducción en n:

  • Caso base: nnn = 000
    ℒ ⁿℒ ᵐ = ℒ ⁰ℒ ᵐ = Λ.ℒ ᵐ = ℒ ᵐ = ℒ ⁰⁺ᵐ = ℒ ⁿ⁺ᵐ
  • Caso inductivo: nnn = k+1k + 1k+1. Supongo que vale para nnn = kkk.
    ℒ ⁿℒ ᵐ = ℒ ᵏ⁺¹ℒ ᵐ = ℒ.ℒ ᵏℒ ᵐ =ᴴᴵ ℒ.ℒ ᵏ⁺ᵐ = ℒ ᵏ⁺¹⁺ᵐ = ℒ ⁿ⁺ᵐ

d. ℒ ⁿ ⊆ ℒ ⁿ⁺¹ para todo 𝑛 ≥ 0

Falso. Contraejemplo: ℒ = {𝑎}. ℒ ⁰ = {𝜆}, ℒ ¹ = {𝑎}, {𝜆} ⊄ {𝑎}.

e. ℒ₁ ⊆ ℒ₂, 𝑛 ≥ 0 ⟹ (ℒ₁)ⁿ ⊆ (ℒ₂)ⁿ

Veradero. Por inducción en n:

  • Caso base: nnn = 000
    (ℒ₁)⁰ = (ℒ₂)⁰ = Λ
  • Caso inductivo: nnn = k+1k + 1k+1. Supongo que vale para nnn = kkk.
    (ℒ₁)ᵏ⁺¹ = (ℒ₁)ᵏ.ℒ₁, (ℒ₂)ᵏ⁺¹ = (ℒ₂)ᵏ.ℒ₂. Por hipótesis inductiva (ℒ₁)ᵏ ⊆ (ℒ₂)ᵏ, y como ℒ₁ ⊆ ℒ₂, se concluye que (ℒ₁)ᵏ⁺¹ ⊆ (ℒ₂)ᵏ⁺¹ ⇔ (ℒ₁)ⁿ ⊆ (ℒ₂)ⁿ.

f. ℒ₁ ⊆ ℒ₂ ⟹ (ℒ₁)∗ ⊆ (ℒ₂)∗

Verdadero. Se deduce de la definición de clausura de Kleene y el inciso anterior.

g. (ℒ ∗)∗ = ℒ ∗

Verdadero. Probemos la doble inclusión:

  • ℒ ∗ ⊆ (ℒ ∗)∗
    Sea 𝑥 ∈ ℒ ∗, 𝑥 es una concatenación de cadenas de ℒ, así que 𝑥 ∈ (ℒ ∗)∗.
  • (ℒ∗)∗ ⊆ ℒ ∗
    Sea 𝑥 ∈ (ℒ ∗)∗, 𝑥 es una concatenación de cadenas 𝑥 de ℒ, así que 𝑥₁𝑥₂…𝑥ₖ con 𝑥ᵢ ∈ ℒ ∗. Cada 𝑥ᵢ es una concatenación de cadenas de ℒ, así que 𝑥 ∈ ℒ ∗.

h. (ℒ ⁺)⁺ = ℒ ∗

Falso. Contraejemplo: ℒ = {𝑎, 𝑏}. 𝜆 ∈ ℒ ∗, pero 𝜆 ∉ (ℒ ⁺)⁺.

i. (ℒ ⁺)∗ = ℒ ∗

Verdadero. Probemos las inclusiones:

  • ℒ ⁺ ⊆ ℒ ∗: ℒ ⁺ son todas las cadenas no vacías formadas por ℒ, y ℒ ∗ incluye todas las cadenas, incluyendo la vacía.
  • (ℒ ⁺)∗ ⊆ ℒ ∗: las cadenas en (ℒ ⁺)∗ son concatenaciones de cadenas no vacías de ℒ, por lo que están en ℒ ∗.
  • ℒ ∗ ⊆ (ℒ ⁺)∗: cualquier cadena de ℒ∗ (incluyendo la vacía) puede ser vista como una concatenación de elementos de (ℒ ⁺)∗.

j. (ℒ₁ ∪ ℒ₂)∗ = (ℒ₁)∗ ∪ (ℒ₂)∗

Falso. Contraejemplo: ℒ₁ = {𝑎}, ℒ₂ = {𝑏}, 𝑎𝑏 ∈ (ℒ₁ ∪ ℒ₂)∗, pero 𝑎𝑏 ∉ (ℒ₁)∗ ∪ (ℒ₂)∗.

k. (ℒ₁ ∩ ℒ₂)∗ = (ℒ₁)∗ ∩ (ℒ₂)∗

Falso. Contraejemplo: ℒ₁ = {𝑎𝑎, 𝑏}, ℒ₂ = {𝑎, 𝑏𝑏}, (ℒ₁ ∩ ℒ₂)∗ = ∅∗ = {𝜆}, mientras que (ℒ₁)∗ ∩ (ℒ₂)∗ contiene por ejemplo 𝑎𝑎𝑏𝑏, entre otras cadenas.

l. (ℒ ²)∗ = ℒ ∗

Falso. Contraejemplo: ℒ = {𝑎}, 𝑎 ∈ ℒ ∗, pero 𝑎 ∉ (ℒ ²)∗.

m. (ℒ ∪ ℒ₂)∗ = ℒ ∗

Falso. Contraejemplo: ℒ = {𝑎}, ℒ₂ = {𝑏}, 𝑎𝑏 ∈ (ℒ ∪ ℒ₂)∗, pero 𝑎𝑏 ∉ ℒ ∗.

n. (ℒ ⁿ)ʳ = (ℒ ʳ)ⁿ para todo 𝑛 ≥ 0

Verdadero. Por inducción en n:

  • Caso base: nnn = 000
    (ℒ ⁿ)ʳ = (ℒ ⁰)ʳ = Λʳ = Λ = (ℒ ʳ)⁰ = (ℒ ʳ)ⁿ
  • Caso inductivo: nnn = k+1k+1k+1. Supongo que vale para nnn = kkk.
    (ℒ ⁿ)ʳ = (ℒ ᵏ⁺¹)ʳ = (ℒ ᵏℒ )ʳ = ℒ ʳ.(ℒ ᵏ)ʳ =ᴴᴵ ℒ ʳ.(ℒ ʳ)ᵏ = (ℒ ʳ)ᵏ⁺¹ = (ℒ ʳ)ⁿ

ñ. (ℒ∗)ʳ = (ℒ ʳ)∗

Verdadero. Se deduce de la definición de clausura de Kleene y el inciso anterior.

Ejercicio 11.
Siendo:
• Sub(ℒ): subcadenas del lenguaje ℒ.
• Ini(ℒ): subcadenas iniciales (prefijos) del lenguaje ℒ.
• Fin(ℒ): subcadenas finales (sufijos) del lenguaje ℒ.
Demostrar que, si ℒ₁ y ℒ₂ son lenguajes:
a. Fin(Fin(ℒ₁)) = Fin(ℒ₁)

Lo probamos por doble inclusión:

  • ⊆) Si 𝑥 ∈ Fin(Fin(ℒ₁)), ∃𝑦 ∈ Fin(ℒ₁) tal que 𝑥 es sufijo de 𝑦, y ∃𝑧 ∈ ℒ₁ tal que 𝑦 es sufijo de 𝑧. Por transitivad 𝑥 es sufijo de 𝑧 ⇒ 𝑥 ∈ Fin(ℒ₁).
  • ⊇) Directo, ya que ℒ₁ ⊆ Fin(ℒ₁).

b. Sub(Sub(ℒ₁)) = Sub(ℒ₁)

Lo probamos por doble inclusión:

  • ⊆) Si 𝑥 ∈ Sub(Sub(ℒ₁)), ∃𝑦 ∈ Sub(ℒ₁) con 𝑥 subcadena de 𝑦, y ∃𝑧 ∈ ℒ₁ con 𝑦 subcadena de 𝑧 ⇒ 𝑥 ∈ Sub(ℒ₁).
  • ⊇) Directo, ya que ℒ₁ ⊆ Sub(ℒ₁).

c. Fin(ℒ₁ℒ₂) = Fin(ℒ₂) ∪ Fin(ℒ₁)ℒ₂

Lo probamos por doble inclusión:

  • ⊆) Para 𝑥 ∈ Fin(ℒ₁ℒ₂), 𝑥 es sufijo de 𝑦𝑧 (𝑦 ∈ ℒ₁, y 𝑧 ∈ ℒ₂). Hay dos casos:
    • Caso 1: Si 𝑥 es sufijo de 𝑧 ⇒ 𝑥 ∈ Fin(ℒ₂).
    • Caso 2: Si no, 𝑥 = 𝑤𝑧 con 𝑤 sufijo de 𝑦 ⇒ 𝑥 ∈ Fin(ℒ₁)ℒ₂.
  • ⊇)
    • Fin(ℒ₂) ⊆ Fin(ℒ₁ℒ₂) (tomar 𝑦 = 𝜆).
    • Fin(ℒ₁)ℒ₂ ⊆ Fin(ℒ₁ℒ₂) (si 𝑤 ∈ Fin(ℒ₁), ∃𝑦 ∈ ℒ₁ con 𝑤 sufijo de 𝑦; luego 𝑤𝑧 es sufijo de 𝑦𝑧).

d. Ini(ℒ₁ ∪ ℒ₂) = Ini(ℒ₁) ∪ Ini(ℒ₂)

Lo probamos por doble inclusión:

  • ⊆) 𝑥 ∈ Ini(ℒ₁ ∪ ℒ₂) ⇒ 𝑥 es prefijo de alguna cadena en ℒ₁ o ℒ₂.
  • ⊇) Trivial por definición de unión.

e. Fin(ℒ₁ ∪ ℒ₂) = Fin(ℒ₁) ∪ Fin(ℒ₂)

Análogo al inciso d, pero para sufijos.

f. Ini(Sub(ℒ₁)) = Sub(Ini(ℒ₁)) = Fin(Sub(ℒ₁)) = Sub(Fin(ℒ₁)) = Sub(ℒ₁)

  • Sub(ℒ₁) ya contiene todos los prefijos, sufijos y subcadenas de ℒ₁.
  • Aplicar Ini, Fin o Sub no añade nada nuevo:
  • Sub(Ini(ℒ₁)) ⊆ Sub(ℒ₁) (las subcadenas de prefijos son subcadenas).
  • Análogo para sufijos.
